# AnnaBot
Automation task bot

## Setup
Create `config_irc.json` in main path.
```json
{
  "server": "<server_name>",
  "gecos": "<gecos_field>",
  "nick": "<bot_visible_nickname>",
  "id": "<bot_identificator>",
  "channels": ["#list", "#of", "#channels"],
  "commands": [
    "======================================",
    "COMMANDS",
    "!command and description",
    "!command and description"
    "======================================"
  ]
}

```
