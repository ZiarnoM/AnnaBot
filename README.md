# AnnaBot
Automation task bot

## Setup
Create `config.json` in main path.
```json
{
  "server": "<server_name>",
  "gecos": "<gecos_field>",
  "nick": "<bot_visible_nickname>",
  "id": "<bot_identificator>",
  "channels": ["#list", "#of", "#channels"],
  "connectionString": "<database connection string>"
}

```
